Le Code

Les lois ancestrales du Village NIRD.

1.

La Sobriété tu viseras

Ne consomme que le nécessaire. Chaque octet compte, chaque requête a un coût. Le vrai résistant est celui qui laisse la plus petite empreinte.

2.

Tes Données tu protégeras

L'Empire se nourrit de tes informations. Utilise le chiffrement, bloque les traceurs, et ne donne jamais ton vrai nom aux machines de Goliath.

3.

Le Savoir tu partageras

Le code est libre, comme la parole. Partage tes outils, tes connaissances et tes découvertes avec le village. L'Open Source est notre arme.

4.

L'Obsolescence tu combattras

Répare ton matériel. Fais durer tes appareils. Ne cède pas à la tentation du "toujours plus neuf" dictée par l'Empire.
